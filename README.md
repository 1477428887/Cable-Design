# 电缆设计结构库软件

**作者：白泽**

一款专为电缆工程设计的离线桌面软件，提供项目文件夹自动生成和电缆参数筛选匹配功能。

## 🚀 快速开始

### 启动软件
双击 `启动电缆设计软件.bat` 文件即可启动

### 📚 文档说明
所有详细文档已整理到 **[文档说明](文档说明/)** 文件夹中：

- **[快速开始指南](文档说明/快速开始指南.md)** - 新用户必读
- **[启动说明](文档说明/启动说明.md)** - 启动和配置说明  
- **[V4.0系统说明](文档说明/V4.0_系统说明.md)** - 完整功能手册
- **[项目清理报告](文档说明/项目清理完成报告.md)** - 项目维护信息

## 📁 项目结构

```
电缆设计系统/
├── 启动电缆设计软件.bat          # 主启动脚本
├── 快速修复依赖.bat              # 依赖修复工具
├── cable_design_system_v4.py     # 主程序文件
├── cable_config.json             # 配置文件
├── cable_products_v4.db          # 产品数据库
├── cable_codes.db                # 代码数据库
├── cable_config_example.json     # 配置示例
├── 文档说明/                     # 📚 所有文档和说明
├── 演示项目文件夹/               # 🎯 演示数据
└── .vscode/                      # 开发配置
```
