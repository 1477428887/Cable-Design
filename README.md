# 电缆设计结构库软件

**作者：白泽**

一款专为海外组电缆报价工作定制的离线桌面软件，提供项目文件夹自动生成、电缆型号智能检索、清单批量解析和项目数据汇总功能。

## 🚀 快速开始

### 第一次使用
1. 双击 `快速修复依赖.bat`，等待自动安装依赖（需联网，只需做一次）
2. 在软件的 **系统设置** 标签页配置三个路径：默认项目文件夹、定额文件夹、技术规范文件夹
3. 之后每次双击 `启动电缆设计软件.bat` 启动

### 系统要求
- Windows 操作系统
- Python 3.6 或更高版本（安装时勾选 "Add Python to PATH"）

## 📋 功能概览

| 标签页 | 功能 |
|--------|------|
| 📁 项目管理 | 一键生成规范命名的项目文件夹，管理历史项目记录 |
| 📋 项目清单列表 | 汇总所有项目清单数据，按月份/项目/电压等级筛选导出 |
| 📋 参数卡片编辑 | 按型号或结构参数检索电缆，查看定额/规范路径，录入新产品 |
| 📊 产品编码管理 | 浏览管理数据库中所有电缆产品，支持多条件筛选和导出 |
| 🤖 智能清单解析 | 批量解析客户原始清单文字，自动识别电压/型号/规格/结构 |
| 系统设置 | 配置项目文件夹、定额库、技术规范库路径 |

## 📁 文件结构

```
电缆设计系统/
├── 启动电缆设计软件.bat          # 日常启动入口
├── 快速修复依赖.bat              # 首次安装依赖
├── cable_design_system_v4.py     # 主程序
├── cable_config.json             # 配置文件（路径、项目记录）
├── cable_products_v4.db          # 电缆产品数据库
├── cable_codes.db                # 型号代码规则库
├── 文档说明/                     # 使用文档
└── 演示项目文件夹/               # 演示数据
```
